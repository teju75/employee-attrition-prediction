# Employee Attrition Prediction

## Project Overview

Employee attrition refers to the situation when employees leave an organization due to various reasons such as job dissatisfaction, better opportunities, or personal factors. High employee turnover can affect productivity and increase recruitment costs for companies.
This project uses machine learning techniques to predict whether an employee is likely to leave the company based on various employee-related factors.

## Dataset

The project uses the **IBM HR Analytics Employee Attrition Dataset**, which contains information about employees such as age, job role, monthly income, job satisfaction, years at the company, and other work-related attributes.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook

## Machine Learning Model

A **Random Forest Classifier** is used to train the model and predict employee attrition.

## Project Steps

1. Data Loading
2. Data Exploration
3. Data Preprocessing
4. Encoding Categorical Variables
5. Train-Test Split
6. Model Training
7. Model Evaluation
8. Employee Attrition Prediction

## Result

The trained model can analyze employee information and predict whether the employee is likely to stay in the company or leave.

## Conclusion

This project demonstrates how machine learning can help organizations identify potential employee attrition and take preventive actions to improve employee retention.
